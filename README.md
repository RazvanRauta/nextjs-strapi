# NextJs and Strapi

Next JS app that is using Redux Toolkit Query to fetch posts via GrpahQL from Strapi CMS.

## Created with

 `1. NextJs 12.0.7`

 `2. Strapi 4.0.0`

 `3. RTK Query 1.7.0`

 `4. GrpahQL 16.1`

 `5. React 17.0.2`

 `6. Tailwindcss 3.0`
 
 `7. React Markdown 7.1.1`

 `8. Graphql Codegen 2.3.0`

 `9. Typescript 4.5.2`
